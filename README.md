# Conversor-.png-.jpg
Esse projeto foi feito no intuito de converter de forma automatizada imagens da pasta selecionada, de .png para .jpg.

Para poder fazer sua utilização é preciso alterar o nome da - pasta origem - para o nome da pasta contendo os arquivos .png.

Uma nova pasta será criada com o nome de "Imagens Convertidas", caso tenha algum arquivo .png na pasta de origem, ele será convertido e acrescido na pasta "Imagens Convertidas".
